# side-ueditor

This repo is explore ueditor integration with Vue 2.x project.

## Quick Start

```bash
pnpm install
pnpm run php:server
pnpm run dev
```

## Links

- 秀米编辑器 demo

```bash
cd php-server
php -S localhost:3000
```
